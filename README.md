## 📊 Projects

### 1. Customer Segmentation Using K-Means Clustering

**Overview:**
Performed customer segmentation on 53,000+ insurance customers using K-Means Clustering to identify distinct customer groups based on Income Level and Premium Amount.

**Key Highlights:**

* Data Cleaning and Exploratory Data Analysis (EDA)
* Feature Scaling using StandardScaler
* Optimal Cluster Selection using Elbow Method
* Customer Segmentation with K-Means Clustering
* Cluster Visualization and Business Recommendations

**Tools Used:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn

**Business Impact:**

* Identified 4 customer segments
* Enabled targeted marketing strategies
* Improved customer retention and cross-selling opportunities

### 2. Superstore Sales Data Analysis

**Overview:**
Conducted Exploratory Data Analysis on Superstore sales data to uncover trends in sales, profitability, customer segments, product categories, and regional performance.

**Key Highlights:**

* Data Cleaning and Validation
* Sales and Profit Analysis
* Regional Performance Evaluation
* Customer Segment Analysis
* Discount Impact on Profitability
* Data Visualization and Business Insights

**Tools Used:** Python, Pandas, Matplotlib, Seaborn

**Business Impact:**

* Identified top-performing product categories
* Analyzed profit contribution by region
* Evaluated customer segment profitability
* Revealed the negative impact of excessive discounts on profits
